Dieses Verzeichnis enthält C#-Projekte, die mit Visual Studio 2005 Beta 1 entwickelt wurden.

Um die Projekte zu starten, müssen Sie die Verbindungszeichenkette von

  "Data Source=192.168.80.128;Initial Catalog=mylibrary;" + 
  "User ID=root;PWD=uranus"

zu

  "Data Source=localhost;Initial Catalog=mylibrary;" + 
  "User ID=root;PWD=yourpassword"

ändern.

-----

These directories contain C# projects developed with Visual Studio 2005 beta 1

To run the examples, you must change the connection string from

  "Data Source=192.168.80.128;Initial Catalog=mylibrary;" + 
  "User ID=root;PWD=uranus"

to

  "Data Source=localhost;Initial Catalog=mylibrary;" + 
  "User ID=root;PWD=yourpassword"
