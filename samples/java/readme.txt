Theses samples are part of the book

  Michael Kofler: MySQL, 35rd edition
  english edition: Apress

This directory contains a few Java samples. You have to change the
username and password to make them work (in the line: conn =
DriverManager.getConnection ...). Then you must compile the files
using JDK 1.3 or higher.

For more information, see chapter 17!
